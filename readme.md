## Build variants

Useful Gradle commands for the lite/full plugin builds:

- `./gradlew jarLite`
- `./gradlew jarFull`
- `./gradlew build` (produces both jars)
- If the Shadow plugin is available: `./gradlew shadowJarLite` / `./gradlew shadowJarFull`
